```sql filename="query.sql"

SELECT users.id, users.first_name, users.last_name, users.email, user_profiles.bio, user_profiles.avatar_url, user_settings.theme_preference, user_settings.notification_settings
FROM users 
LEFT JOIN user_profiles ON users.id = user_profiles.user_id 
LEFT JOIN user_settings ON users.id = user_settings.user_id 
WHERE users.created_at BETWEEN '2024-01-01 00:00:00' AND '2024-12-31 23:59:59'
```

### Testing Instructions

1. Open this article in a browser
2. Scroll horizontally in any of the code blocks above
3. Verify that line numbers remain visible and "stick" to the left side
4. Test in both light and dark modes
5. Test on different screen sizes

The line numbers should maintain their position on the left side of the code block while the code content scrolls horizontally.
